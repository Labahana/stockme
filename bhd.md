| Factor | Weight | What You Control |
|--------|--------|----------------|

| **Subtitle/tagline** | High | 70 characters, keyword-rich |
| **Description** | Medium | Keyword density, feature bullets |
| **Reviews** | High | Volume + rating + recency |
| **Install velocity** | High | Downloads in first 30 days |
| **Screenshots** | Medium | Alt text, feature visibility |
| **Categories** | Medium | "Inventory management" |
| **Built for Shopify badge** | High | Search ranking boost |

## The Review Flywheel (Critical for App Store SEO)

Shopify's algorithm weights **review velocity** heavily.

| Reviews in First 30 Days | Search Ranking |
|-------------------------|----------------|
| 0–5 | Buried on page 3+ |
| 10–20 | Page 2 |
| 30–50 | Page 1, bottom |
| 50–100 | Page 1, top 5 |
| 100+ | Featured |

**How to get reviews:**

1. **Day 3 after install:** In-app banner: "Loving StockMe? Leave a review — it helps other Stocky refugees find us."
2. **Day 7:** Personal email: "Hey [Name], saw you created 3 POs this week. Would you mind leaving a quick review? Takes 30 seconds."
3. **Day 14:** If no review, another nudge: "Your trial ends in 7 days. A review unlocks an extra week free."
4. **Respond to every review** — positive and negative. Shopify's algorithm sees engagement.
